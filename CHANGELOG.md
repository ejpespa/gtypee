## [1.0.0] - 2026-02-21

### Added
- Initial release of gtypee
- Gmail commands (send, list, search, drafts, threads, labels, filters, signatures)
- Drive commands (ls, search, upload, download, permissions, comments, revisions)
- Calendar commands (events, create, update, conflicts)
- Workspace Admin commands (users, groups, org units, devices, reports)
- Docs, Sheets, Slides commands
- Tasks, Forms, Contacts, People, Chat, Classroom, Groups, Keep, Apps Script commands
- OAuth 2.0 and Service Account authentication
- JSON output support for all commands
- Desire-path aliases for common commands
